Simple music composition tool.
Write lyrics, create chord progressions, and play around with quick arrangements.

Current modes:
- Lyrics
- Audio
-   Record audio
- Compose
  -   Chords
  -   Sampler
  -   Bass
  -   Beat
  -   Sequencer
  -   Melody
  -   Progressions
-   Mixer
-   Notes
-   Arranger
